# CashKey Property Group - Project Outline

## Website Structure & File Organization

### Core Pages (4 HTML Files)
1. **index.html** - Homepage with lead generation focus
2. **about.html** - Company story and team information  
3. **services.html** - Detailed service offerings and process
4. **contact.html** - Contact forms and service area information

### Supporting Files
- **main.js** - Core JavaScript functionality and interactions
- **resources/** - Directory for images and media assets

## Page-by-Page Content Strategy

### 1. Homepage (index.html)
**Primary Goal**: Maximum lead generation and conversion

**Sections**:
- **Navigation Bar**: Logo, main menu, phone number, "Get Offer" CTA
- **Hero Section**: 
  - Compelling headline with typewriter animation: "Sell Your House Fast for Cash"
  - Professional hero image of a welcoming home
  - Primary CTA button leading to lead capture form
  - Trust indicators (BBB rating, years in business, homes purchased)
- **Quick Lead Capture Form**: 
  - Address input with instant offer estimate
  - Multi-step form with smooth transitions
  - Progress indicator and validation
- **Value Proposition Section**:
  - "Why Choose CashKey" with animated counters
  - Key benefits: Speed, No Repairs, No Fees, Certainty
- **Process Overview**:
  - 3-step visual process (Contact → Evaluate → Close)
  - Interactive timeline with hover effects
- **Social Proof Carousel**:
  - Customer testimonials with photos
  - Before/after property examples
  - Auto-rotating with manual controls
- **Property Calculator**:
  - Interactive tool for offer estimation
  - Slider inputs for property details
  - Real-time calculation display
- **FAQ Section**:
  - Common questions about cash home buying
  - Expandable accordion format
- **Footer**: Contact info, service areas, copyright

## Interactive Components Implementation

### 1. Lead Capture Form System
- **Technology**: Vanilla JavaScript with validation
- **Features**: Multi-step progression, real-time validation, progress indicator
- **Integration**: Form submission handling and lead management

### 2. Property Evaluation Calculator
- **Technology**: JavaScript with Matter.js for interactive sliders
- **Features**: Real-time calculations, visual feedback, result display
- **Data**: Market-based pricing algorithms and repair cost estimates

### 3. Testimonials Carousel
- **Technology**: Splide.js with custom styling
- **Features**: Auto-rotation, manual navigation, responsive design
- **Content**: Customer photos, quotes, property details

### 4. Service Area Map
- **Technology**: Interactive SVG map with hover effects
- **Features**: Clickable regions, coverage confirmation, area details
- **Integration**: Form pre-population based on selected area
